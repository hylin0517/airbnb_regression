# Airbnb Fair Price Estimation

This project builds regression models to estimate a reasonable nightly Airbnb price from listing characteristics such as location, room type, capacity, amenities, availability, and review signals.

The business goal is not only to predict price, but to compare an observed listing price against the model's fair-price estimate so a user can judge whether a listing appears overpriced or like a good deal.

The main analysis lives in [notebooks/airbnb_2026_price_prediction.ipynb](notebooks/airbnb_2026_price_prediction.ipynb), using historical Inside Airbnb New York City listings data from January and February 2026.
